*Quellcode:* <br>
Was ist das? <br> Quellcode ist einfach der Text, den du in einer höheren Programmiersprache schreibst. 
<br> Zum Beispiel: 
python 
x = 5 y = 10 print(x + y)  
Das ist Quellcode. Der Computer versteht es erst, wenn es in Maschinensprache übersetzt wird.
